## Key Findings

- **OpenWork** is the only tool that combines GUI-first UX + team skills sharing + MCP + open source + free desktop
- **Hermes Desktop** is the automation backbone (cron, multi-platform gateway) but not GUI-first for non-tech
- **Kimi** has good desktop apps but closed ecosystem — no MCP, no skills hub
- **Perplexity** and **Lindy** are not agent platforms — wrong tool for dev/agency work
- **Nebula.ai** domain is for sale — product shut down or rebranded
- **Windsurf** merged with Devin (Cognition) in 2026

## Evaluation Criteria for Agency AI Stack

Must-haves for Ornevo-type agencies:
1. **GUI-first for non-technical users** (25% weight)
2. **Shared skills/knowledge hub** (20% weight)
3. **MCP compatibility** (15% weight)
4. **Cost** (15% weight)
5. **GitHub org integration** (10% weight)
6. **Multi-OS** (5% weight)
7. **Scalability** (5% weight)
8. **Privacy** (5% weight)
